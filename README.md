# Kaggle Journey

A personal collection of notebooks, experiments, and utilities created while participating in Kaggle competitions and learning data science / ML workflows.

## Why I'm Doing This

I want to build practical, portfolio-ready machine learning skills by solving real problems on Kaggle. This repo tracks my progress, experiments, and learnings in a structured and reproducible way.

- Focus: end-to-end ML projects (EDA → features → models → evaluation → reporting)
- Outcome: a body of work that demonstrates applied ML skills and systematic thinking

## Commitment

- Time: **8–10 hours per week**
- Start date: **2025-03-01**  
- Cadence:
  - Weekly: at least one focused study / experiment block
  - Monthly: consolidate learnings, clean up best notebooks, and update this README

## 12-Month Goal

By 12 months from the start date, I aim to:

	•	Achieve Kaggle Expert minimum.
	•	Complete at least 8 competitions fully.
	•	Earn minimum 1 medal (bronze or higher).
	•	Build 3 portfolio-grade notebooks.
